# 햇빛 추적 지능형 빨래 건조대 개발 (아두이노 & 센서 제어)

## 📌 프로젝트 개요
- **기간**: 2023.10 – 2024.05 (졸업 작품)
- **팀 구성**: 7명 (본인 역할: 자료조사, CATIA 3D 모델링/어셈블리 검증, 아두이노 센서 데이터 처리 및 모터 제어 알고리즘 구현, 졸업 논문 작성)
- **핵심 목표**: 고정형 건조대의 한계를 극복하기 위해, CDS 조도센서 배열로 햇빛이 가장 강한 곳을 실시간 추적하고 리니어 스텝 모터를 제어하여 자동으로 최적의 위치로 이동하는 스마트 시스템을 설계 및 제작합니다.

## 🛠 기술 스택
- **Hardware & Control**: Arduino (C++), CDS 조도센서 (6개 배열), 포토인터럽터 (위치 피드백), 스텝 모터 (HCMotor 라이브러리)
- **Design & Manufacturing**: CATIA 3D Modeling, Ender 5 Pro 3D 프린터
- **Interface**: Python Serial Communication (데이터 로깅용)

## 💡 핵심 설계 및 문제 해결 (Technical Focus)
1. **조도센서 배열 기반 최댓값 탐색 알고리즘**
   - 레일 방향을 따라 6개의 CDS 조도센서를 배열하고, `analogRead`를 통해 들어오는 아날로그 값(0~1023)을 실시간 비교 연산하여 가장 조도가 높은 구역의 인덱스를 도출하는 제어 로직을 설계했습니다.
2. **포토인터럽터를 활용한 클로즈드 루프(Closed-Loop) 제어**
   - 초기 모터 구동 시 오픈 루프 제어로 인해 정확한 정지 위치를 잡지 못하거나 탈조가 일어날 수 있는 리스크가 있었습니다.
   - 각 정지 지점에 포토인터럽터를 배치하여 모터가 구동하는 동안 실시간 위치 피드백(값 950 초과 시 정지)을 받아 정지하는 **클로즈드 루프 제어 구조**를 적용하여 센서 데이터와 실제 건조대 이동 위치의 일치성을 확보했습니다.
3. **속도 최적화 및 구조적 경량화**
   - 초기 구동 시 리니어 모터의 이송 속도가 느린 문제가 발생했으나, 코드를 통해 제어 펄스 및 Duty Cycle을 최적화하여 해결했습니다.
   - CATIA로 설계한 프레임을 3D 프린팅(PLA)으로 출력하여 **기존 금속 프레임 대비 60% 이상 무게를 경량화**함과 동시에 빨래 하중을 견딜 수 있는 구조적 강성을 확보했습니다.

## 📈 주요 성과
- **기능 검증**: 손전등 조사를 통한 실시간 햇빛 추적 및 모터 자동 이동 하드웨어 테스트 완수
- **학업 성과**: 아두이노 C++ 제어 알고리즘 및 구조 검증 내용을 바탕으로 학부 졸업 논문 작성 및 기계공학부 졸업(2026.02)
